Selamat Datang!

Kita akan belajar cara membuat website dari nol!

Langkah 1: HTML

HTML adalah dasar dari sebuah website. Buat file index.html dan tulis struktur dasar HTML.

Langkah 2: CSS

Gunakan CSS untuk mempercantik tampilan website. Buat file style.css dan hubungkan ke HTML.

Langkah 3: JavaScript

Tambahkan interaksi dengan JavaScript. Buat file script.js dan gunakan event listener.

Langkah 4: Jalankan Website

Buka file HTML di browser dan lihat hasilnya!

Langkah 5: Hosting

Gunakan layanan gratis seperti GitHub Pages atau Netlify untuk membuat website bisa diakses semua orang.

Tips Terakhir

Pastikan kode kamu rapi dan gunakan desain yang responsif agar website nyaman di berbagai perangkat.

Selesai!

Selamat! Sekarang kamu sudah bisa membuat website sendiri!
